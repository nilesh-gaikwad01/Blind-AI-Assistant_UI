# Blind AI Assistant Mobile App

## Features

* Voice Guidance
* Multi-language Support
* Object Detection
* Face Recognition
* Indoor Navigation
* SOS Emergency Feature

## Setup

```bash
flutter pub get
```

Run:

```bash
flutter run
```

Backend URL:

```text
http://YOUR_SERVER_IP:5000/detect
```
