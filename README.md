# Blind AI Assistant

An AI-powered assistive application for visually impaired individuals.

## Technologies

### Frontend

* Flutter
* Flutter TTS
* Camera

### Backend

* Python
* Flask
* YOLOv8
* OpenCV
* Face Recognition

## Features

✅ Object Detection

✅ Face Recognition

✅ Navigation Guidance

✅ Multi-language Support

🚧 Currency Recognition

🚧 OCR Reading

🚧 Emergency SOS

## Architecture

Mobile App
→ Flask API
→ AI Models
→ Voice Guidance
